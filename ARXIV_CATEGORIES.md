## 使用示例

在命令行中使用时，可以通过 `--category` 参数指定领域：

```bash
# 搜索人工智能领域的论文
python main.py --category cs.AI --days 7 --max-results 5

# 搜索机器学习领域的论文
python main.py --category cs.LG --days 7 --max-results 5

# 搜索计算机视觉领域的论文
python main.py --category cs.CV --days 7 --max-results 5

# 搜索量子物理领域的论文
python main.py --category quant-ph --days 7 --max-results 5
```

## 常用领域推荐

如果你对以下研究方向感兴趣，推荐使用这些分类：

- **人工智能与机器学习**：`cs.AI`, `cs.LG`, `cs.NE`, `cs.CL`
- **计算机视觉**：`cs.CV`, `cs.MM`
- **机器人学**：`cs.RO`, `cs.AI`
- **自然语言处理**：`cs.CL`, `cs.AI`
- **网络安全**：`cs.CR`, `cs.NI`
- **量子计算**：`quant-ph`, `cs.ET`
- **深度学习**：`cs.LG`, `cs.NE`, `cs.CV`
- **数据挖掘**：`cs.DB`, `cs.IR`, `cs.LG`

更多详细信息请访问 [arXiv 官网](https://arxiv.org/category_taxonomy)
